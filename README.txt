This project is a movie trailer website that will help you to choose a movie.

Instruction:
    * For adding a new movie please make a Media class object
    * Add this class in entertainment.py file with additional attributes and add it to list
    * After that fresh_tomatoes.py will automatically add it on the Movie Trailer Site
How to run:
    To run the application type "python entertainment.py" on system's command line and after that
    open the fresh_tomatoes.html to look on Movie Trailer Website
